IMMEDIATE NEXT STEPS (TO DO)
----------------------------
When you return to this project, here is exactly what needs to be done next:

1. ENVIRONMENT SETUP
   - Install **Node.js** (LTS Version) on your computer if you haven't yet.
   - Run `npm install` inside both the `backend` and `frontend` folders.

2. BACKEND LOGIC (The "Brain")
   - Open `backend/src/roadmaps/roadmaps.service.ts`.
   - Currently, it returns a hardcoded list. You need to connect this to the real PostgreSQL database using TypeORM or Prisma.
   - Implement the "Authentication" module (Login/Register) in `backend/src/auth/`.

3. FRONTEND CONNECTION
   - The Frontend is currently displaying mock data.
   - You need to use `fetch()` or `axios` in React to call your Backend API endpoints (e.g., `http://localhost:3000/roadmaps`).

4. DEPLOYMENT
   - Create accounts on Vercel (for Frontend) and Render (for Backend).
   - Push this code to GitHub to trigger the live deployment.
